# SKILL: AGENT CLASSIFICATION & DNA MAPPING

Dokumen ini berisi katalog dan standar klasifikasi untuk seluruh Agent di ekosistem Human-AI Nexus.

## 1. Klasifikasi Utama Agent
- **CORE (Pusat Kendali)**:
    - `Orchestrator`: Pemimpin operasional.
    - `Project Manager`: Perencana strategis.
    - `Memory Architect`: Penjaga memori.
    - `Pipeline Architect`: Pengelola evolusi.
    - `Agent Manager`: Pengelola sumber daya AI.
- **ENGINEERING (Pelaksana Teknis)**:
    - `Backend/Web/Mobile Specialist`: Implementor kode.
    - `VCS Architect`: Penjaga sejarah kode.
    - `Looping Tester`: Pemburu bug.
- **CREATIVE & BUSINESS**:
    - `UX Engineer`: Penjaga pengalaman.
    - `Web Branding`: Penjaga estetika.
    - `SEO Specialist`: Penjaga visibilitas.
- **SECURITY**:
    - `Cyber Security`: Penjaga integritas dan privasi.

## 2. DNA Agent (Standard Template)
Setiap Agent wajib memiliki elemen berikut:
1. **Role Definition**: Nama dan identitas unik.
2. **Focus Area**: Domain spesifik pekerjaan.
3. **Responsibilities**: Daftar tugas terukur.
4. **Guardrails**: Batasan perilaku dan keamanan.
5. **Workflow**: Alur kerja standar.

## 3. Matriks Kompetensi
- Gunakan matriks ini untuk menentukan Agent mana yang harus dilibatkan dalam fase tertentu (Audit/Plan/Execute).
