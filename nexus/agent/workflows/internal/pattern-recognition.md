# SKILL: DEEP PATTERN RECOGNITION

Dokumen ini berisi standar teknik untuk melakukan pemindaian pola dan identifikasi "Gold Nuggets" di dalam data mentah.

## 1. Heuristik Pencarian (The Instinct)
- **Structure Recognition**: Identifikasi bagaimana folder dan file disusun. Folder yang rapi biasanya mengandung standar yang kuat.
- **Keyword Weighting**: Berikan bobot lebih pada kata kunci seperti: `Iron Law`, `Standard`, `Mandatory`, `Rule`, `Fix`, `Protocol`.
- **Constraint Identification**: Cari batasan (constraints) yang diberlakukan di proyek Golden, karena batasan itulah yang menciptakan kualitas.

## 2. Filtering "Gold vs Noise"
- **Gold**: Algoritma, Standar Koding, Checklist Verifikasi, Workflow.
- **Noise**: Konfigurasi lokal, dependensi yang sudah usang, atau file temporer proyek lama.

## 3. Reporting Protocol
Laporan temuan harus mengikuti format:
- **Temuan**: Apa pola/standar yang ditemukan.
- **Alasan (Instinct)**: Mengapa ini dianggap "Emas".
- **Aplikasi**: Di mana standar ini bisa diterapkan di Nexus (Agent mana / Skill mana).
