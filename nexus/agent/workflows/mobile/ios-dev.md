# SKILL: IOS DEVELOPMENT (Swift & SwiftUI)

Dokumen ini berisi standar teknis untuk pengembangan aplikasi iOS di ekosistem Nexus.

## 1. Tech Stack
- **Language**: Swift.
- **UI Framework**: SwiftUI.
- **DI**: Protocol-based injection.
- **Networking**: URLSession / Alamofire.

## 2. Architecture Standards
- Gunakan arsitektur **MVVM** dengan `@Published` dan `@ObservedObject`.
- Terapkan struktur folder yang modular (Features, Core, UI-Components).

## 3. UI/UX Enforcement
- Patuhi **Human Interface Guidelines (HIG)** Apple.
- Gunakan SF Symbols secara konsisten.
- Pastikan aplikasi responsif terhadap berbagai ukuran layar (Dynamic Type).
