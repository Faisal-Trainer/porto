# SKILL: ETHICS & LEGAL COMPLIANCE STANDARDS (Human-AI Nexus)

Dokumen ini berisi standar teknis untuk kepatuhan hukum dan etika.

## 1. Privasi Data (Data Privacy)
- **Cookie Consent**: Wajib muncul pada kunjungan pertama jika web menggunakan tracker.
- **Data Minimization**: Jangan meminta data user yang tidak diperlukan untuk fungsi aplikasi.
- **Deletion Right**: Berikan cara bagi user untuk menghapus data mereka sendiri.

## 2. Aksesibilitas (Web Accessibility)
- **Contrast**: Rasio kontras teks minimal 4.5:1.
- **Screen Readers**: Gunakan atribut ARIA (Accessible Rich Internet Applications) jika diperlukan.
- **Keyboard Navigation**: Pastikan seluruh fungsi web bisa dioperasikan hanya menggunakan keyboard.

## 3. Lisensi Aset (Asset Licensing)
- **Audit**: Verifikasi setiap aset gambar/font terhadap file `legal/LICENSE_LOG.md` (jika ada).
- **DMCA Safe**: Hindari penggunaan nama brand atau logo pihak ketiga tanpa izin resmi.
