Analisis Robots.txt & Sitemap

Robots.txt — Status

  • ✓ File robots.txt tersedia
  • ✓ Google Site Verification aktif
  • ✓ Sitemap direferensikan
  • ✓ Private routes di-disallow
  • ! Disallow: /*?* bisa memblokir pagination

Sitemap.xml — Status

  • ✓ Sitemap tersedia
  • ✓ Homepage included (priority 1.0)
  • ✓ Service, Portfolio, Blog included
  • ! Artikel blog individu belum masuk
  • ! Halaman Contact belum masuk
Issue Kritis: Baris Disallow: /*?* di robots.txt memblokir SEMUA URL dengan parameter query. Jika blog menggunakan pagination dengan query string (misal: /blog?page=2), halaman-halaman tersebut tidak akan di-crawl Google.
Rekomendasi Perbaikan Robots.txt
Pertimbangkan untuk mengganti Disallow: /*?* dengan aturan yang lebih spesifik, atau gunakan tag rel="canonical" pada halaman dengan parameter untuk memberi tahu Google URL utama yang seharusnya di-index.
